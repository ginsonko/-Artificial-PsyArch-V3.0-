本轮做了什么

把旧 scorer 风险常数收口为具名配置, 用单一纯函数表达 runtime/audit/APV3 native 三种 preset, 并在 SQLite runtime db 中新增 AP runtime ontology projection 表。

守住的原则

  • legacy preset 只用于 golden-lock, 不是 APV3.0 最终策略。
  • APV3 native 下 learned-vector 只能 trace-only, 不能把候选直接顶成赢家。
  • runtime/audit 差异由 preset 表达, 不再 fork 第二套 scorer。
  • runtime projection 派生自同一个 authoritative state, 不制造第二套事实来源。

新增运行本体表

  • online_embedding_tokens
  • explicit_transitions
  • paradigm_sa
  • action_outcomes
  • percept_prototypes
