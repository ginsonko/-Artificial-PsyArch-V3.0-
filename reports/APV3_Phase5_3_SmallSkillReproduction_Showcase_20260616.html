新教学协议

1. Successor
问候、成语通过重复 cue → reply 学后继。
2. Aggregation
颜色对象通过多回复样例形成槽位。
3. Process
数学过程学习结构，不接隐藏计算器。
4. Focus Fill
当前 focus token 作为工作记忆积木填槽。
5. Recall Only
验收只给 cue，不给 teacher reply。

失败归因与修正

  • 数学槽位一开始被历史 relation 压过当前 focus，已修正为当前 focus token 优先作为工作记忆积木。
  • 未知 cue 一开始能靠 context 召回问候，已修正为 Bn recall 必须有最小 cue grasp。
  • 两处修正都是通用 AP 读数边界，不是技能关键词规则。
