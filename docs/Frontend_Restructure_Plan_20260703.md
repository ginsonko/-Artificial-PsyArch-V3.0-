# 前端4分页面重构计划

## 需求确认

### 1/ xiaoyi语音包
- **结论**: xiaoyi是浏览器Web Speech API内置音色,非独立语音包文件
- **含义**: 不需上传语音包,小白浏览器自带。但不是所有系统有xiaoyi
- **方案**: 配置页让用户选可用音色,默认尝试xiaoyi→退回其他中文音色

### 2/ 首页 (默认打开)
- 对话框+输入区(图片/音频/教学/发送/闲时tick/清空)
- 按钮: 多模态识图示例(原体验示例)/小学数学泛化学习示例(原体验数学)/加载范式种子/加载预热包
- 右侧: 内心画面/想法云/内心音频(大一些不挤)
- 引导介绍: 先点什么再点什么怎么测试教学纠正用法
- 连续闲时/刷新记忆

### 3/ 详情页
- 运行状态/学习生命周期验收/tick回放/草稿格/短期结构流/审计曲线/记忆
- 每个卡片加介绍说明(做什么/怎么理解/有什么用)
- 内容可长,滚动条向下,整齐美观大方
- **竖式过程可视化**: 用户特别要求能看到列竖式的过程

### 4/ 配置页
- 本地语音音色选择(默认xiaoyi→退回中文)
- 其他可配置项(查代码确认有哪些)
- 每个配置有注释/推荐值/应用/恢复默认
