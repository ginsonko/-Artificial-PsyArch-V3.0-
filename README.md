# APV3.0test

APV3.0test 是人工心灵架构 Artificial PsyArch 的 V3.0 开放对话底座仓库。

它提供一个可直接运行的本地工作台，包含真实对话、教学、视觉输入、音频输入、记忆包导入导出、理论读本和演示入口。普通使用不需要额外安装依赖，双击 `启动APV3体验台.bat` 即可打开工作台；也可以直接运行前端入口启动本地服务后访问工作台页面。

## 这份仓库里有什么

- `apv3test/`：运行时、Web 服务、前端静态页
- `scripts/`：验收、教学、发布、资产处理脚本
- `tests/`：回归测试与守护测试
- `docs/`：理论读本、设计说明、验收报告
- `data/`：本地运行数据、课程资产、工作台数据库

## 如何启动

1. 双击 `启动APV3体验台.bat`
2. 浏览器会自动打开 `http://127.0.0.1:8765/phase20_7`
3. 在工作台里直接体验对话、图片、音频、教学、记忆包与理论页

理论页入口：

- 工作台内的“理论”入口
- 直接访问 `http://127.0.0.1:8765/phase20_7_theory.html`

## 你会看到什么

- 真实的中文开放对话
- 教学后可召回的记忆
- 图片与音频输入
- 记忆包的导入、导出、卸载
- 理论读本与 QA 说明
- 运行时 tick 回放、草稿、审计和记忆视图
