核心链路

1. Observation
自然教学或 LLM 标准教学产生同构 SA 经验。
2. Dirty Bucket
只更新所属 cue/case 桶，不全局重算。
3. Local Discovery
局部调用 DP / coherence / Viterbi。
4. ParadigmSA
范式作为状态池一等对象进入运行态。
5. Persistence
SQLite 保存观察、统计和角色转移证据。

本轮吸收的纠偏

  • 相似 context 只通过 promoted learned vector 泛化，未 promoted token 不参与相似泛化。
  • 范式惩罚暴露门复用行动后果同类半衰期和近因衰减。
  • 统计池只存细节，ParadigmSA 必须同步进入 paradigms 和 state_field_items。
  • 自然教学与 LLM 教学的 provenance 可以不同，但 runtime 竞争对象必须等价。
