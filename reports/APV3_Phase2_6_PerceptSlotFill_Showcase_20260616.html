阶段结论

APV3.0test 现在能把稳定 percept token 接入范式槽填充候选。最小黄色苹果链路不再靠目标中文串, 而是由当前 percept token、范式槽和草稿游标共同产生逐 token 草稿候选。

本轮新增

  • ParadigmSlotFiller: 通用槽填充器。
  • FillCandidate: 槽候选及来源证据。
  • DraftCandidate: 带 text_visible_draft_token/v1 游标 metadata 的逐 token 草稿候选。

关键探针

conf 0.031606 roles ['slot', 'slot']
drafts [('percept::yellow', 'slot', ''), ('percept::apple', 'slot', 'percept::yellow')]
joined percept::yellowpercept::apple

输出是 percept token 序列, 不是 黄色苹果 目标串。第二个 token 带 previous_prefix, 后续行动器仍应一 tick 执行一个赢家 token。
