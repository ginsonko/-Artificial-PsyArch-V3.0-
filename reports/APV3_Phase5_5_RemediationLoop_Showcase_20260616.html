边界
非答案补丁

补习只写教学 evidence，不改 runtime 输出规则。

闭环路径

Train
运行原始教学 episode。
Validate
只给 cue / context / 合法当前 focus。
Diagnose
记录 bn_not_recalled 等 failure_kind。
Remediate
生成 AP-native CurriculumTeachingStep。
Validate
补习后重新验证。

成功样例

初始验证:
cue = 你 好
expected = 我 在
emitted = <empty>
diagnosis = bn_not_recalled

补习:
stage = remediate:bn_not_recalled
case_name = skill_greeting
cue = 你 好
reply = 我 在
teaching step count = min_support

补习后:
cue = 你 好
emitted = 我 在
diagnosis = success

教学等价

natural final emitted = 我 在
llm_standard_teacher final emitted = 我 在
natural final focus_pid = llm final focus_pid
state 中没有 llm_policy
LLM 可以是教师，但不能是学生侧运行时策略。两种教学来源最终都落成同构 AP-native evidence。
